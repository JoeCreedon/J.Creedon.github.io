Bookmarker App

Project Details

This was another JavaScript challenge I gave myself. I used an "if" function to test whether the user enters a valid website. If they do not, an alert pops up to prompt them to. If they do, the input is held in local storage and pushed an array on the HTML page as output. I used JASON to "stringify" the data and Bootstrap as a CSS3 theme.
